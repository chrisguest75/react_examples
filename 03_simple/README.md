# README

Demonstrate another simple example from `freecodecamp`.  

## 🏠 Build

```sh
# configure
nvm use
npm install
```

## ⚡️ Run

```sh
# start the app
npm start
```

## Creation

```sh
# in repo root
cd $(git root)

# if npx is not working
npx clear-npx-cache

# 
nvm install --lts
npx create-react-app 03_simple
```
