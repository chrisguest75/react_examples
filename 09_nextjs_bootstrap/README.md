# README

Demonstrate using bootstrap in NextJS.  

## Install

```sh
# switch versions
nvm use  

# install modules
npm install

# run
npm run dev
```

Now click the hyperlink.  
 
## Created

```sh
npx create-next-app --typescript 09_nextjs_bootstrap


```
