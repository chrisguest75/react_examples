## Install

```sh
# switch versions
nvm use  

# install modules
npm install

# run
npm run dev
```

Now click the hyperlink.  

## How to run

```sh
nvm use
npm install

# use typescript compiler
npm run tsc -- --version  

# run targets
npm run dev
npm run test
npm run lint
```

## Created

```sh
npx create-next-app --typescript 13_shadcn
npx shadcn@latest init

# adding navigation and icons
npx shadcn@latest add navigation-menu
npm i @radix-ui/react-icons
```
