# README
Demonstrate the most basic `react` app

NOTES: 
* Uses JSX instead of JS
* Uses `className` instead of `class` for css

## Creation

```sh
# in repo root
cd $(git root)
npx create-react-app 01_helloworld 

cd 01_helloworld 
npm start
```

Adding icons
```sh
npm install react-icons --save             
```
