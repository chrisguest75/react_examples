# README

Build a `connect4` game.  

## 🏠 Build

```sh
# configure
nvm use
npm install
```

## 🧪 Test

```sh
# run tests
npm test
```

## ⚡️ Run

```sh
# start the app
npm start
```

## Creation

```sh
# in repo root
cd $(git root)
npx create-react-app 05_connect4 --template typescript
cd 05_connect4

npm start  
```
