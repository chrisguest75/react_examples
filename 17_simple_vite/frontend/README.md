# VITE

A simple Vite project.

## Create

To recreate the project, you can use the following commands:

```sh
npm create vite@latest
```

## Install

To start the project, run:

```sh
nvm use
#or 
nvm alias default v20.19.0

npm install
```

## Run

```sh
npm run dev
```
